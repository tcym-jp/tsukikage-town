# 月影町 Codex長期実装パッケージ

このフォルダ一式を、月影町専用のGitリポジトリ直下へ配置してください。

## 目的

Codexに一度の大きな指示を送り、ブラウザで遊べる完成品の探索アドベンチャーゲームを、設計・実装・検証・改善まで連続して進めてもらうための仕様一式です。

仮タイトル：

**月影町 ― 十三番目の回覧板**

## 使い方

1. このフォルダ内のファイルをリポジトリ直下へ配置する。
2. Codexをそのリポジトリで開く。
3. 長時間作業向けの高い推論設定を選べる場合は選択する。
4. パッケージ導入、ローカルコマンド、ブラウザテストを許可する。
5. `CODEX_START_PROMPT.md` の本文を一度だけ送る。
6. `/goal` が使えない場合は、先頭の `/goal` を削除して通常指示として送る。

## ファイル

- `AGENTS.md`：常時守る開発ルール
- `PRODUCT_SPEC.md`：製品仕様と技術仕様
- `STORY_BIBLE.md`：世界観、登場人物、7日間の物語
- `UX_VISUAL_SPEC.md`：3D、画面、操作、アクセシビリティ
- `ACCEPTANCE_TESTS.md`：完成条件と受入試験
- `PLAN.md`：Codexが更新する計画・進捗
- `DOCUMENTATION.md`：Codexが更新する設計判断・テスト・再開情報
- `CODEX_START_PROMPT.md`：最初に送る一回分の指示

## 重要

- 目的は「利用枠を消費すること」ではなく、公開可能な完成品を作ることです。
- 広いだけの3D空間ではなく、転入から3種類以上の結末まで遊べる一本の体験を優先します。
- 有料API、ログイン、サーバーDB、広告、課金、生成AI APIは初期版に入れません。
- 外部素材は権利が明確なものだけ使用します。可能な限り、コード、SVG、Canvas、Web Audio、プリミティブ3Dで自作します。
- GitHubへのpushや本番デプロイは、認証と明示的許可がある場合だけ行います。
